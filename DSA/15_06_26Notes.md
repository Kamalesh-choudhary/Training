### Q1] Pairs With Sum = 0
Given an array you need to return the pair of integers whose sum is 0. But all the pairs following such condition should be given out as in a sorted array format.

        [2,5,6,1,0,-1,-6,-5,0,-2]

#### Ans:
        [(-6, 6), (-5, 5), (-2, 2), (0, 0)]

#### Algorithm:
    Step 1: Sort the array
    Step 2: Set l = 0, r = n-1
    Step 3: While l < r:

        Step 3.1 : Calculate total = arr[l] + arr[r]
        Step 3.2 : If total == 0:

            Step 3.2.1 : Push {arr[l], arr[r]} to answer
            Step 3.2.2 : Save left = arr[l], right = arr[r]
            Step 3.2.3 : Skip all duplicates on the left
            Step 3.2.4 : Skip all duplicates on the right

        Step 3.3 : If total < 0: increment l
        Step 3.4 : If total > 0: decrement r
    Step 4: Return answer

#### Code:
````cpp
class Solution {
public:
    vector<vector<int>> getPairs(vector<int>& arr) {
        sort(arr.begin(),arr.end());
        vector<vector<int>> ans;
        int n = arr.size();
        int l = 0, r = n-1;
        int left = 0, right = 0;
        while(l<r){
            int total = arr[l]+arr[r];
            if (total == 0){
                ans.push_back({arr[l],arr[r]});
                left = arr[l];
                right = arr[r];
                while(l<r && arr[l] == left) l+=1;
                while(l<r && arr[r] == right) r-=1;
            }
            if (total < 0) l+=1;
            if (total > 0) r-=1;
        }
        return ans;
    }
};
````

### Q2] Count Pairs with Sum = Target
Given an array return the Count of pair of integers whose sum is equal to a given number.

        [1,5,7,-1,5], target = 6

#### Ans:
        3
#### Algorithm:
    Step 1: Sort the array
    Step 2: Set l = 0, r = n-1, cnt = 0
    Step 3: While l < r:

        Step 3.1 : Calculate total = arr[l] + arr[r]
        Step 3.2 : If total == target:
            Step 3.2.1: If arr[l] == arr[r] (all remaining elements are equal):
                Calculate tmp = r - l + 1
                Add tmp * (tmp-1) / 2 to cnt (combinations formula)
                Break

            Step 3.2.2: Else:
                Count consecutive duplicates on left → leftcnt
                Count consecutive duplicates on right → rightcnt
                Add leftcnt * rightcnt to cnt

        Step 3.3 : If total < target: increment l
        Step 3.4 : If total > target: decrement r


    Step 4: Return cnt

#### Code:
```cpp
class Solution {
public:
    int countPairs(vector<int> &arr, int target) {
        sort(arr.begin(),arr.end());
        int cnt = 0;
        int n = arr.size();
        int l = 0, r = n-1;
        while(l<r){
            int total = arr[l]+arr[r];
            if (total == target){
                if(arr[l] == arr[r]){
                    int tmp = r-l+1;
                    cnt += tmp*(tmp-1)/2;
                    break;
                } else {
                    int left = arr[l], right = arr[r];
                    int leftcnt = 0, rightcnt = 0;
                    while(l<r && arr[l] == left){ leftcnt+=1; l+=1; }
                    while(l<=r && arr[r] == right){ rightcnt+=1; r-=1; }
                    cnt += leftcnt*rightcnt;
                }
            }
            if (total < target) l+=1;
            if (total > target) r-=1;
        }
        return cnt;
    }
};
````

### Q3] Maximum Water storage Between Reservoir Walls

A civil engineer is designing a water storage system along a riverbank. A series of vertical reservoir walls have been erected at equally spaced positions along the bank. Each wall has a specific height. The engineer wants to select exactly two walls such that the water trapped between them (along with the ground as the base) is maximized.

The amount of water stored between two walls at positions i and j (where i < j) is determined by the shorter of the two walls multiplied by the distance between them: min(walls[i], walls[j]) * (j - i).

Given an integer array walls of length n representing the height of each wall, return the maximum amount of water that can be stored between any two walls.

Note: The container cannot be slanted — water level is always determined by the shorter wall.
#### Input Format:
- Line1: An interger n
- Line2: n space-seperated integers representing the height of the wall.
#### Output Format:
print Single integer:- The maximum volume of water that can be stored between any two walls.
#### Algorithm:
    Step 1: Set l = 0, r = n-1, ans = INT_MIN
    Step 2: While l < r:
        Step 2.1: Calculate area = min(walls[l], walls[r]) * (r - l)
        Step 2.2: Update ans = max(ans, area)
        Step 2.3: If walls[l] >= walls[r]: decrement r
        Step 2.4: Else: increment l

    Step 4: Return ans
#### Code:
```C++
#include <iostream>
#include <vector>
#include <bits/stdc++.h>
using namespace std;

int maxWaterStorage(vector<int>& walls) {
    //===== Write Your Logic Here =====
    int n = walls.size();
    int l = 0;
    int r = n-1;
    int ans = INT_MIN;
    while(l<r){
        int area = min(walls[l],walls[r])*(r-l);
        ans = max(ans,area);
        if (walls[l] >= walls[r]){
            r-=1;
        }
        else{
            l+=1;
        }
    }
    return ans;

}

int main() {
    int n;
    cin >> n;
    vector<int> walls(n);
    for (int i = 0; i < n; i++) cin >> walls[i];
    cout << maxWaterStorage(walls) << endl;
    return 0;
}
```

### Q4] Four Signal Frequencies Matching target
A radio telescope station records signal frequency deviations from multiple antennas throughout an observation session. Scientists want to identify all unique combinations of exactly four antenna readings whose combined frequency deviation matches a specific target value.

Given an integer array signals of length n and an integer target, return all unique quadruplets [signals[a], signals[b], signals[c], signals[d]] such that a, b, c, and d are distinct indices and signals[a] + signals[b] + signals[c] + signals[d] == target.

You may return the answer in any order.
#### Input Format:
- Line1: An integer n
- Line2: n space-seperated integers representing the height of the each wall.
#### Output Format:
Print a single integer- The maximum volume of the water that can be stored between any two walls.
#### Constraints:
    2<= n<= 10^5
    0<= walls[i]<=10^4
#### Algorithm:
    Step 1: Sort the array
    Step 2: Loop i from 0 to n-3:

        Step 2.1: Skip duplicates for i
        Step 2.2: Loop j from i+1 to n-2:

            Step 2.2.1: Skip duplicates for j
            Step 2.2.2: Set l = j+1, r = n-1
            Step 2.2.3: While l < r:

                Calculate total = signals[i] + signals[j] + signals[l] + signals[r]
                If total < target: increment l
                If total > target: decrement r
                If total == target:

                Push quadruplet to answer
                Skip duplicates on left
                Skip duplicates on right

    Step 3: Return answer
#### Code:
```C++
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<vector<int>> findQuadruplets(vector<int>& signals, int target) {
    //===== Write Your Logic Here =====4
    sort(signals.begin(),signals.end());
    vector<vector<int>> ans;
    int n = signals.size();
    for(int i=0;i<n-3;i++){
        if (i>0 && signals[i] == signals[i-1])continue;
        for(int j=i+1;j<n-2;j++){
            if (j>i+1 && signals[j] == signals[j-1])continue;
            int l = j+1;
            int r = n-1;
            while(l<r){
                int total = signals[i]+signals[j]+signals[l]+signals[r];
                if (total < target)l++;
                else if (total > target)r--;
                else{
                    ans.push_back({signals[i],signals[j],signals[l],signals[r]});
                    int left = signals[l];
                    int right = signals[r];
                    while(l<r && signals[l] == left)l++;
                    while(l<r && signals[r] == right)r--;
                }
            }
        }
    }
    return ans;
}

int main() {
    int n, target;
    cin >> n >> target;
    vector<int> signals(n);
    for (int i = 0; i < n; i++) cin >> signals[i];

    vector<vector<int>> quads = findQuadruplets(signals, target);
    if (quads.empty()) {
        cout << -1 << endl;
    } else {
        for (auto& q : quads)
            cout << q[0] << " " << q[1] << " " << q[2] << " " << q[3] << "\n";
    }
    return 0;
}
```

### Q5] Zero Sum Sensor Triplets
A weather monitoring station collects readings from multiple sensors throughout the day. Each reading represents a deviation from the baseline temperature — positive values indicate above-baseline and negative values indicate below-baseline readings.

The station's data analyst wants to identify all unique groups of three sensor readings (from different sensors) whose deviations perfectly cancel each other out (i.e., sum to zero). This helps detect balanced anomaly patterns in the data.

Given an integer array readings of size n, return all unique triplets [readings[i], readings[j], readings[k]] such that i != j, i != k, j != k, and readings[i] + readings[j] + readings[k] == 0.

The result must not contain duplicate triplets.
#### Input Format:
    7
    [-2,0,1,3,-1,-2,2]
#### Output Format:
    -2 -1 3
    -2 0 2
    -1 0 1
#### Consrtaints:
    3<= n <= 3000
    -10^5 <= readings[i] <= 10^5
#### Algorithm:
    Step 1: Sort the array
    Step 2: Loop i from 0 to n-2:

        Step 2.1: Skip duplicates for i
        Step 2.2: Set l = i+1, r = n-1
        Step 2.3: While l < r:

            Step 2.3.1: Calculate total = readings[i] + readings[l] + readings[r]
            Step 2.3.2: If total < 0: increment l
            Step 2.3.3: If total > 0: decrement r
            Step 2.3.4: If total == 0:

                Push triplet to answer
                Skip duplicates on left
                Skip duplicates on right

    Step 3: Return answer
#### Code:

### Q6] Closest cargo load to Target weight
A logistics company is loading cargo containers onto a ship. Each container has a weight value (which can be negative due to buoyancy adjustments). The ship has an ideal target load capacity, and the crew must select exactly three containers from the available list such that their combined weight is as close as possible to the target capacity.

Given an integer array weights of length n and an integer target, find three integers at distinct indices in weights whose sum is closest to target. Return that closest sum.

You may assume that each input has exactly one solution.
#### Input Format
    5 10
    [-3,2,8,5,-1]
#### Output Format
    10
#### Constraints
    3 <= n <= 500
    -1000 <= weights[i] <= 1000
    -10^4 <= target <= 10^4
#### Algorithm: 
    Step 1: Sort the array
    Step 2: Initialize mx = weights[0] + weights[1] + weights[2]
    Step 3: Loop i from 0 to n-1:

        Step 3.1: Set l = i+1, r = n-1
        Step 3.2: While l < r:

            Step 3.2.1: Calculate total = weights[i] + weights[l] + weights[r]
            Step 3.2.2: If total == target: return target immediately
            Step 3.2.3: If |mx - target| > |total - target|: update mx = total
            Step 3.2.4: If total < target: increment l
            Step 3.2.5: Else: decrement r

    Step 4: Return mx
#### Code:
```C++
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>
#include <bits/stdc++.h>
using namespace std;

int closestSum(vector<int>& weights, int target) {
    //===== Write Your Logic Here =====
    int n = weights.size();
    sort(weights.begin(),weights.end());
    int mx = weights[0]+weights[1]+weights[2];
    for(int i=0;i<n;i++){
        int l = i+1;
        int r = n-1;
        while(l<r){
            int total = weights[i]+weights[l]+weights[r];
            if (total == target){
                return target;
            }
            else{
                if (abs(mx-target) > abs(total-target)) mx = total;
            }
            if (total < target)l++;
            else r--;
        }
    }
    return mx;
}

int main() {
    int n, target;
    cin >> n >> target;
    vector<int> weights(n);
    for (int i = 0; i < n; i++) cin >> weights[i];
    cout << closestSum(weights, target) << endl;
    return 0;
}
```
